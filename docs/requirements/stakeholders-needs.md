## Короткий огляд продукту

Система аналізу медіа-контенту являє собою комплекс програмного забезпечення, який передбачає системи для моніторингу максимально можливої кількості публічних джерел, збору, систематизування, перевірки достовірності та аналізу інформації, а також побудови інфографіки. 

## Функціональність

Функціальні вимоги реалізовані у вигляді різних інтерфейсів для різних користувацьких ролей (користувач, адміністратор).

Інтерфейс користувача включає у себе наступні функції:

- Форма для авторизації та реєстрації
- Персональний кабінет
- Форма для виконання запиту з можливісту перегляду історії власних запитів
- Модуль перегляду інфографіки та інших діаграм 

Інтерфейс для адміністратора передбачає можливість перегляду профілів користувачів та списку усіх запитів і перегляду даних для відладки.

## Практичність

Основною вимогою є простий та інтуїтивно зрозумілий інтерфейс, його лаконічність та впізнаваність.
